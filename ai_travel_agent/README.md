# ✈️ AI Travel Agent

Get a complete, personalised day-by-day travel itinerary for any destination — with accommodation, food, transport, and budget breakdown.

## How it works
Answer 5 questions (destination, days, budget, interests, travel style) and GPT-4o-mini generates a detailed, specific itinerary.

## Quickstart
```bash
pip install -r requirements.txt
cp ../.env.example .env
python agent.py
```

## What you get
- Day-by-day schedule (morning/afternoon/evening)
- Accommodation picks by neighbourhood + price range
- Local dishes + restaurant suggestions
- Transport tips
- Full budget breakdown
- Cultural tips and packing list

## Stack
- `openai` — GPT-4o-mini with a rich travel-expert system prompt
